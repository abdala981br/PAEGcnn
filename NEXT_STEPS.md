# Proximos Passos

Os proximos passos do projeto seguem tres frentes principais.

Primeiro, vamos continuar realizando testes com novos modelos, ampliando a base de comparacao e buscando alternativas com melhor desempenho para os cenarios avaliados.

Depois, vamos montar um dataset mais bem trabalhado, com mais consistencia e melhor qualidade para apoiar testes, ajustes e futuras validacoes de forma mais confiavel.

Por fim, vamos estabelecer a comunicacao e a utilizacao dos modelos via API, permitindo integrar a inferencia a outros sistemas e estruturar um fluxo de uso mais escalavel.
